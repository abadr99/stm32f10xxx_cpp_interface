Detailed Description

template<stm32::type::ConnectionType CT>
class stm32::dev::hal::buzzer::Buzzer< CT >

Class representing a Buzzer controller.

This template class allows controlling a buzzer, including turning it on, off, and toggling its state. It also allows querying the current state of the buzzer.

Template Parameters
CTThe connection type for the buzzer, typically defined as stm32::type::ConnectionType.
